# Source Conflict Escalation

When internal materials conflict, do not blend the values. Identify the conflict, use stable facts if possible, and escalate the disputed detail.

## Priority order

1. Current admissions operations guidance
2. Current catalog policy or catalog program page for official requirements
3. Current public program page for student-facing overview
4. Older one-page sheets or archived training documents

## Staff answer pattern

- State that documents conflict on the specific detail.
- Use current operations guidance for talking points when it is clearly newer.
- Name the documents to verify.
- Send the issue to admissions operations or the program owner for confirmation.
