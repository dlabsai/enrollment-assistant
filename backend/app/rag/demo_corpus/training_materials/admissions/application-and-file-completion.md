# Application and File Completion

An application is ready for review only when the required file items for the selected program are present. Requirements vary by program level and can change by catalog year.

## Standard staff guidance

- Use https://apply.demo-university.example.edu/ for the application link.
- Ask which program the student selected before listing required file items.
- Tell students to request official transcripts early.
- Use unofficial transcripts only for early advising, not final admission or transfer-credit decisions.
- Escalate uncertain program requirements to admissions operations rather than improvising.

## Transcript wording

Students should send official electronic transcripts to transcripts@demo-university.example.edu when the sending school supports electronic delivery. Paper transcripts should use the mailing address provided by the advisor or records team.
