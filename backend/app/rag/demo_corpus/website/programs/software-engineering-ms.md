# Software Engineering, MS

The Software Engineering, MS program focuses on architecture, scalable systems, quality, cloud concepts, and technical leadership.

## Program overview

Students study how to design and maintain complex systems while communicating tradeoffs across technical and organizational teams.

## What you will study

- Software architecture
- Cloud and distributed systems
- Quality engineering
- Secure development practices
- Technical project leadership

## Format and pacing

The program is offered online. Students should expect team-oriented projects and architecture-focused writing.

## Related programs

Artificial Intelligence, MS; Cybersecurity, MS; and Data Analytics Graduate Certificate may be relevant.
