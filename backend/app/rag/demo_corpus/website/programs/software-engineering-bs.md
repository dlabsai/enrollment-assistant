# Software Engineering, BS

The Software Engineering, BS program prepares students to design, build, test, and maintain software systems.

## Program overview

Students learn programming, algorithms, design, testing, databases, and collaborative project workflows.

## What you will study

- Programming and software design
- Data structures and algorithms
- Databases and web systems
- Testing and quality practices
- Team project delivery

## Format and pacing

Online and campus options are available. Students with transfer credit in programming should request review before choosing a course sequence.

## Related programs

Artificial Intelligence, BS; Cybersecurity, BS; and Software Engineering, MS are common comparisons.
